# 🚀 Dhruv Product Data Management System

A Java Console Based Mini Project developed using Core Java concepts to simulate a Product Data Management (PDM) system similar to enterprise applications.

---<img width="358" height="228" alt="image" src="https://github.com/user-attachments/assets/5b6eee2d-5649-40d1-b9e0-c0ff15e42213" />


## 📌 Project Overview

This application allows users to:

- Secure Login Authentication
- Create Master Data
- Search Existing Data
- Delete Records
- Auto Number Generation
- Console Based Navigation Menus

---

## 🛠 Technologies Used

- Java
- Core Java
- OOP Concepts
- Collections (ArrayList)
- Exception Handling
- Eclipse IDE
- Git & GitHub

---

## 📂 Modules Included

### 🔐 Login Module
- Username & Password Validation
- Multiple Login Attempts

### ➕ Create Data Module
- Part
- Document
- Manufacturer
- Manufacturer Part

### 🔍 Search Data Module
- View all created records

### ❌ Delete Data Module
- Delete records by Number / ID

---

## 🔢 Auto Number Formats

- Part Number → 100-262-100
- Document Number → DOC-200
- Manufacturer ID → MFR-300
- Manufacturer Part → MFP-400
